Logs are located in the container folders here - var/lib/docker/containers/

Calculating the space occupied by logs -  

```bash
sudo sh -c "du -ch /var/lib/docker/containers/*/*-json.log"
```

Clearing logs  
```bash
sudo sh -c "truncate -s 0 /var/lib/docker/containers/*/*-json.log"
```
